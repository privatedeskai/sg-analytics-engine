Как работает повторный запуск шаблона
Быстрый режим
Структура данных не изменилась. Система запускает сохранённый Python скрипт напрямую через E2B — без участия AI модели.
Скорость: ~5 секунд
Стоимость: E2B только
Когда: регулярные отчёты
Выбирается автоматически
Полный режим
Структура данных изменилась или запрос требует новой интерпретации. Kimi K2.6 строит анализ заново через 10 итераций.
Скорость: ~60 секунд
Стоимость: Kimi K2.6 + E2B
Когда: новые данные или структура
Выбирается автоматически
Пользователь нажимает одну кнопку "Запустить" — система сама определяет режим на основе сравнения схемы данных
